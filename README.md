# Student Management System

A Spring Boot-based Student Management System developed using Java and Spring Framework.

## Features

- Student CRUD Operations
- Course Management
- Spring Data JPA
- MySQL Database
- Bean Validation
- Thymeleaf Templates
- Maven Build
- RESTful Architecture

## Tech Stack

- Java 17
- Spring Boot
- Spring Data JPA
- Hibernate
- MySQL
- Thymeleaf
- Maven
- HTML
- CSS

## Tools

- Eclipse IDE
- Git
- GitHub
- Postman
